# Software Project Plugin

Software development lifecycle plugin for OrqaStudio — delivery planning with milestones, epics, tasks, research discovery, and roadmap views.

## Usage

See the [CLI reference documentation](documentation/cli-reference.md) or run `orqa --help`.

## Development

```bash
npm install
npm run build
```
