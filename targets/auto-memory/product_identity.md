name: Product identity
description: OrqaStudio is a clarity engine for structured thinking, not an AI development tool
type: project
---

OrqaStudio is an opinionated clarity engine that helps people turn messy situations into structured understanding and evolving plans. It is NOT an AI development tool — it is cognitive infrastructure for structured reasoning that uses AI as its reasoning partner.

**Why:** Every feature decision must be evaluated against "does this improve structured thinking?" not "does this make AI-assisted coding better?" Software development is the first domain because governance can integrate directly into dev workflows, but the product serves anyone doing complex work: product managers, researchers, consultants, solo operators.

**How to apply:** When implementing features or making design decisions, check against the vision (VISION-4893db55) and pillars, not against what competing AI dev tools do. OrqaStudio is closer to "cognitive infrastructure" than to "GitHub Copilot."
