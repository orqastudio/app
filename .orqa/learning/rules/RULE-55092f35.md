Every Svelte component directory in `libs/svelte-components/src/` (both `pure/` and `connected/`) must contain at least one `.stories.ts` file.

## Rationale

Components without stories are invisible in Storybook, cannot be visually reviewed by designers or other developers, and drift from the design system over time. Stories serve as living documentation and visual regression test anchors.

## What constitutes a violation

A component directory that contains one or more `.svelte` files but zero `.stories.ts` files.

## Enforcement

### Pre-commit check (`orqa check component-stories`)

The CLI `orqa check` command should include a `component-stories` subcheck that:

1. Scans `libs/svelte-components/src/pure/` and `libs/svelte-components/src/connected/` for component directories (depth 1)
2. For each directory containing at least one `.svelte` file, verifies at least one `.stories.ts` file exists
3. Reports violations with the directory path and a suggestion for the story filename
4. Exits non-zero if any violations are found

### Behavioural (agent guidance)

When creating a new component in `libs/svelte-components/`, always create a `.stories.ts` file alongside it. Follow the existing pattern:

- `Meta` with `title: "Pure/ComponentName"` or `title: "Connected/ComponentName"`
- `tags: ["autodocs"]` for automatic documentation generation
- `argTypes` for all variant, size, and boolean props
- One `Story` export per variant/state combination

## Story file conventions

```typescript
import type { Meta, StoryObj } from "@storybook/svelte";
import MyComponent from "./MyComponent.svelte";

const meta = {
  title: "Pure/MyComponent",  // or "Connected/MyComponent"
  component: MyComponent,
  tags: ["autodocs"],
  argTypes: {
    variant: { control: "select", options: ["default", "secondary"] },
    disabled: { control: "boolean" },
  },
} satisfies Meta;

export default meta;
type Story = StoryObj;

export const Default: Story = { args: { variant: "default" } };
export const Secondary: Story = { args: { variant: "secondary" } };
export const Disabled: Story = { args: { disabled: true } };
```
