# Tauri Plugin Setup — Agent Reference

## Installation

```bash
orqa plugin install @orqastudio/plugin-tauri
```

## Installer Actions

1. Detects sub-projects with Rust/Tauri code
2. Recommends which sub-projects should receive the plugin
3. Verifies Rust stable toolchain is installed
4. Generates `clippy.toml` and `.rustfmt.toml` from coding standards rules

## Generated Config

- `clippy.toml` — lint levels from enforcement entries
- `.rustfmt.toml` — formatting rules from enforcement entries
- Cargo test integration via `orqa check` and `orqa test`

No npm dependencies — Rust tools ship with the toolchain.

## Organisation Mode

At org level, plugin detects Rust sub-projects by scanning for `Cargo.toml` files. AI recommends which apply. Each gets config generated from org-level rules. Sub-projects can override with tracked rationale.
