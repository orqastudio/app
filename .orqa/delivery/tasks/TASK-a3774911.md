## Scope

### manifest.json schema change

Current per-plugin entry:
```json
{
  "version": "0.1.4-dev",
  "installed_at": "...",
  "files": [".orqa/process/rules/RULE-xyz.md"]
}
```

New per-plugin entry:
```json
{
  "version": "0.1.4-dev",
  "installed_at": "...",
  "files": {
    ".orqa/process/rules/RULE-xyz.md": {
      "sourceHash": "sha256:abc123...",
      "installedHash": "sha256:abc123..."
    }
  }
}
```
