## What

This is the integration validation task. After all moves and renames in TASK-1 through TASK-8, sweep the entire codebase for stale references and validate the full install pipeline.

## Knowledge Needed

- Read all plugin manifests: `plugins/*/orqa-plugin.json`
- Read `.orqa/manifest.json` for installed state
- Read `.orqa/prompt-registry.json` for prompt-registry plugin entries
- Read `.orqa/workflows/*.resolved.yaml` for resolved workflow state
- Read `libs/cli/src/commands/install.ts` for BUILD_ORDER and plugin references
- Grep for old names: `plugin-software-project`, `plugin-agile-governance`, `plugins/software/`, `plugins/agile-governance/`, `workflow: delivery`

## Agent Role

Implementer — cross-reference updates and validation commands. This task requires shell access for running `orqa install`, `tsc`, and `cargo clippy`.

## Steps

1. Grep the ENTIRE repo for stale references:
   - old name: `plugin-software-project` (now `plugin-software-kanban`)
   - old name: `plugin-agile-governance` (now `plugin-agile-methodology`)
   - `plugins/software/` (as a path reference, not history)
   - `plugins/agile-governance/` (as a path reference, not history)
   - `workflow: delivery` or `contributes_to.workflow: delivery`
   - Any other old plugin/workflow names
2. Fix every stale reference found
3. Update all manifest `provides` arrays to reflect the new plugin structure:
   - Each plugin's manifest accurately lists what it provides
   - No plugin claims to provide types/workflows that have been moved
4. Update `.orqa/prompt-registry.json` if it references old plugin names
5. Run `orqa install` from a clean state and verify 0 errors
6. Inspect `.orqa/workflows/*.resolved.yaml` to verify:
   - `agile-methodology` workflow exists and is resolved
   - `planning-methodology` point filled by agile-planning
   - `implementation-workflow` point filled by software-kanban
   - `review-process` point filled by agile-review
   - `learning-pipeline` point filled by core (optional, should be present)
   - `discovery-artifacts` point filled by agile-discovery (optional, should be present)
7. Run `npx tsc --noEmit` for `libs/cli`
8. Run `cargo clippy -- -D warnings`
