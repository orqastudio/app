## Current Ports

| Service | Current Port | Proposed |
| --------- | ------------- | ---------- |
| Git server (Docker) | 3030 | 10030 |
| Vite dev server | 1420 | 10420 |
| Dev dashboard | 3001 | 10001 |
| Daemon | 3002 | 10002 |
| Sync bridge | 3001 | 10003 |
| SSH (git) | 222 | 10222 |

## Why

Port 3000/3001 is the most commonly used dev port. The git server already clashed with another Docker container on port 3000 during this session. Moving everything above 10000 makes OrqaStudio a good citizen alongside other local development.
