# CLI Tools Plugin

OrqaStudio plugin that bundles the orqa CLI with skills for validation, version management, plugin management, and dev environment maintenance.

## Usage

See the [CLI reference documentation](documentation/cli-reference.md) or run `orqa --help`.

## Development

```bash
npm install
npm run build
```
