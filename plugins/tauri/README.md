# Tauri Development

OrqaStudio plugin for Rust and Tauri development — clippy, rustfmt, cargo test infrastructure with rule-driven coding standards enforcement.

## Installation

```bash
orqa plugin install @orqastudio/plugin-tauri
```

## Usage

Coding standards are defined as OrqaStudio rules. The plugin generates tool config files from those rules. Run `orqa check` to enforce.
