# Systems Thinking Plugin

OrqaStudio plugin providing systems thinking methodology — second-order effects analysis, diagnostic methodology, and restructuring methodology. Teaches agents to trace consequences, find root causes, and refactor with awareness of the whole system.

## What It Does

- **Systems thinking** — trains agents to reason about second-order effects and unintended consequences
- **Diagnostic methodology** — structured root cause analysis before any fix is applied
- **Restructuring methodology** — safe refactoring patterns that preserve system invariants
- **Architectural evaluation** — framework for evaluating architecture decisions against system-wide constraints
- **Composability** — principles for designing components that remain correct when combined

## Installation

```bash
orqa plugin install @orqastudio/plugin-systems-thinking
```

## Development
