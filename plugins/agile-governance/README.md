# Agile Governance Plugin

OrqaStudio plugin providing the core governance model — rules, decisions, lessons, knowledge, agents, and foundational principles. Defines artifact types and frontmatter schemas that drive structured thinking and enforcement.

## What It Does

- **Artifact schemas** — defines the canonical types: vision, pillar, persona, idea, decision, rule, lesson, knowledge, agent, doc, and pivot
- **Relationship model** — governs how artifacts connect: upholds, drives, governs, enforces, codifies, informs, teaches, and more
- **Foundational agents** — includes core governance agents for orchestration and review
- **Status transitions** — enforces valid lifecycle transitions for every artifact type

## Installation

```bash
orqa plugin install @orqastudio/plugin-agile-governance
```

## Development

```bash
npm install
npm run build
```
