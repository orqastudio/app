# Svelte Development

OrqaStudio plugin for TypeScript and Svelte development — ESLint, svelte-check, Vitest infrastructure with rule-driven coding standards enforcement.

## Installation

```bash
orqa plugin install @orqastudio/plugin-svelte
```

## Usage

Coding standards are defined as OrqaStudio rules. The plugin generates tool config files from those rules. Run `orqa check` to enforce.
